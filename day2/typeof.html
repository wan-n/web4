자료형(간단하게) 1. 숫자 : 정수, 실수 2. 문자 : "", '' 3. 논리 : true, false 4. 배열 : ['', '', '' ...] 데이터를 묶는다. 5. 객체 : 한 가지만 담을 수 있는 원시형과는 달리 다양한 데이터를 담을 수 있다. key : value로 이루어진 프로퍼티를 여러 개 넣을 수 있다. user라는 객체를 만들고자 할 때, let user = { name: "이름", age: 20 }; user.name 형식으로 데이터를 가져올 수 있다. = : 대입 == : 같은지 비교 === : 자료형까지 비교 자바스크립트 -> 5 == '5' (true), 5 === '5' (false) 오류 undefined : 값이 없을 때 null : 값이 잘못되었을 때
